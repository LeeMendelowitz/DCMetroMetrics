DC Metro Metrics

DC Metro Metrics is a project dedicated to collecting and sharing publicly available data related to the DC Metrorail system.

This website compiles data which is not available anywhere else, including:

  • A directory of all 588 escalators in the DC Metrorail system
  • Performance history for each escalator in the system.
  • Escalator reliability rankings
  • Escalator availability by station
  • Crowdsourced Twitter reports of #wmata #hotcar's.

This website and its features are a work in progress. If you have any questions or comments, or would like to get involved, please contact me at info@dcmetrometrics.com.
